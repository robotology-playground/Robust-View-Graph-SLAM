자신의 MathWorks 계정으로 로그인

MathWorks® 계정의 전자 메일 주소와 암호를 입력하고 다음을 클릭합니다. 활성화 응용 프로그램이 MathWorks에 액세스하여 설치를 진행하는 사용자의 계정과 연결된 라이선스를 가져옵니다.

MathWorks 계정이 없는 경우 계정을 만듭니다(활성화 키가 필요함) 옵션을 선택하고 다음을 클릭합니다.

라이선스 파일이 이미 있으면 라이선스 파일의 전체 경로를 파일 이름을 포함하여 입력하십시오 옵션을 선택하고 파일의 전체 경로를 지정한 후 다음을 클릭합니다. 라이선스 파일이 설치를 진행하는 사용자가 실행할 수 있는 제품을 식별합니다. 라이선스 파일이 아직 없다면 라이선스 담당 관리자에게 문의하십시오. 라이선스 파일을 지정하면 활성화 응용 프로그램은 프로세스의 모든 후속 단계를 건너뛰고 활성화 완료 대화 상자로 바로 이동합니다.
