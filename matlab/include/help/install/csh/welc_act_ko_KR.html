설치 활성화

활성화는 MathWorks® 제품의 라이선스가 부여되었는지를 확인하는 프로세스입니다. 이 프로세스에서는 라이선스의 유효성을 검사하고 사용자가 구매한 라이선스에서 선택 가능한 각 설치 옵션에서 허가된 시스템 활성화 가능 횟수를 초과하지 않는지 확인합니다.

설치를 진행하는 사용자가 설치 시 자신의 MathWorks 계정으로 로그인한 상태로 활성화 프로세스를 진행하면, 로그인 세션이 계속 유지된 채로 활성화가 진행됩니다. 다음을 클릭하여 활성화를 진행합니다.

설치 시 자신의 MathWorks 계정으로 로그인하지 않았거나 활성화 응용 프로그램을 독립적으로 시작한 경우, 활성화를 자동으로 진행할지 또는 수동으로 진행할지 선택해야 합니다. 인터넷에 연결되어 있는 경우 인터넷을 사용하여 자동으로 활성화(권장) 옵션을 선택된 채로 둡니다. 설치 후에 바로 활성화하는 것이 MATLAB®을 사용할 수 있는 가장 빠른 방법입니다.

인터넷에 연결되어 있지 않은 경우 인터넷 없이 수동으로 활성화를 선택하십시오. 이 옵션을 선택한 경우 수동으로 활성화하려면 라이선스 파일이 필요합니다. 라이선스 파일이 설치를 진행하는 사용자가 실행할 수 있는 제품을 식별합니다. 라이선스 파일이 아직 없다면 라이선스 담당 관리자에게 문의하여 이 파일을 받으십시오.
