ファイル インストール キー

ファイル インストール キーによって、インストール可能な製品が識別されます。これはインターネットに接続していない場合は必須です。MATLAB® 管理者は、MathWorks® Web サイトのライセンス センターからファイル インストール キーを取得できます。

ファイル インストール キーを使用する場合、次の "いずれか" からインストーラーを実行する必要があります。

  • MATLAB DVD 1。

  • インストーラーと製品ファイルを格納するフォルダー。

    MATLAB 管理者はインストーラーを実行して必要なファイルをこのフォルダーにダウンロードできます。
